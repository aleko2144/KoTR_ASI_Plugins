## KoTR_HelicoptersMod (02.09.2025)
Version 1.00</br>
A modification that allows you to change the positions of helicopters in the game and add new ones.

### KoTR_HelicoptersMod.ini:
```
[COPTER_<index>]
module =  db - name of b3d-file from env directory
room   =  room_db_023 - name of b3d room, where helicopter will placed
posX   =  2650.0
posY   = -2400.0
posZ   =  550.0
radius =  600.0 - radius of the helicopter's coverage area beyond which it will not fly
```

The index starts from 1.
